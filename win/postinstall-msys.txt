MSYS
 
 1 Add "C:\msys\bin" to PATH
    - Right-click "My Computer" and select "Properties"
    - Click the "Advanced" tab
    - Click the "Environment Variables" button
    - Under "System Variables", select "Path" then click "Edit" button
    - Append ";C:\msys\bin" to end
    - Make sure it is after mingw entry
 
 2 Make shortcut on Desktop to "msys.bat"
    - Can Ctrl+Shift+LMB drag to create shortcut
    - Make sure "Start in" field is "C:\msys\bin"
    - Set icon to "C:\msys\msys.ico" with "Change Icon" button
 
 3 Add "alias ls='ls --color'" to "C:\msys\etc\profile"
